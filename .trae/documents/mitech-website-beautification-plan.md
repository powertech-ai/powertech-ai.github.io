# Mitech 网站美化计划

## 目标
将现有简洁的网站改造成具有科技范的专业企业官网，符合 Apple 组织账号审核要求。

## 设计方向
- **风格**：科技感、现代感、专业简洁
- **配色**：深蓝/黑色系为主，搭配科技蓝 accent 色
- **特点**：渐变背景、玻璃拟态效果、流畅动画

## 具体实施步骤

### 1. 创建共享样式文件
- 创建 `styles.css` 包含：
  - CSS 变量定义（主色、辅色、文字色）
  - 全局样式重置
  - 科技感背景效果（渐变、网格、粒子）
  - 玻璃拟态卡片样式
  - 按钮、链接 hover 动画
  - 响应式布局

### 2. 更新 index.html（首页）
- 引入新样式文件
- 添加科技感动画背景（渐变流动效果）
- 重新设计导航栏（玻璃拟态效果）
- 美化标题区域（大字体、渐变文字）
- 添加服务卡片展示区
- 添加页脚

### 3. 更新 about.html（关于我们）
- 引入新样式文件
- 添加页面头部背景
- 美化公司介绍布局
- 经营范围使用卡片/标签形式展示
- 添加时间线或数据展示区域

### 4. 更新 contact.html（联系我们）
- 引入新样式文件
- 添加地图占位区域或地址卡片
- 美化联系信息展示
- 添加联系表单（可选）

## 技术方案
- **CSS 特性**：CSS Grid、Flexbox、CSS 变量、backdrop-filter
- **动画**：CSS 关键帧动画、渐变流动效果
- **图标**：使用 SVG 或 Font Awesome CDN
- **响应式**：移动端适配

## 文件变更清单
1. 新建：`styles.css` - 全局样式
2. 修改：`index.html` - 首页重构
3. 修改：`about.html` - 关于页面重构
4. 修改：`contact.html` - 联系页面重构

## 设计预览

### 配色方案
- 主背景：`#0a0a0f` → `#1a1a2e` 渐变
- 强调色：`#00d4ff`（科技蓝）
- 文字主色：`#ffffff`
- 文字辅色：`#a0a0b0`

### 特效
- 背景：动态渐变流动 + 细微网格
- 卡片：玻璃拟态（backdrop-filter: blur）
- 按钮：hover 发光效果
- 文字：渐变文字效果

## 验收标准
- [ ] 三个页面风格统一
- [ ] 具有明显科技感和现代感
- [ ] 在桌面和移动端都有良好显示
- [ ] 加载流畅，动画不卡顿
- [ ] 符合 Apple 组织账号官网要求
